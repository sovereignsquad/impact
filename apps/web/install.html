Install IMPACT

IMPACT runs as a CLI on your machine. macOS is the supported primary path; Linux is partial; Windows is experimental — see support matrix.

Path C (npm): npm install -g @impact/cli is not the live primary install until #34 is closed with publish, verify, smoke, and evidence. Until then, use Path B below.

Path B — from source (verified today)

Requires Git, Node.js 20+, and npm.

git clone https://github.com/sovereignsquad/impact.git
cd impact
npm ci
npm run build
npm install -g ./apps/cli

Optional: pin to release tag v0.3.0 before npm ci if you want a known release.

Path C — npm registry (after #34)

When the package is public and verified:

npm install -g @impact/cli

Track progress on issue #34.

Prerequisites

  • Node.js 20+
  • Git (for Path B)
  • macOS 13+ recommended for host probes

After install — expected outputs

mkdir -p ./reports
impact scan --no-submit -o ./reports
open ./reports/impact-report.html

Artifacts: impact-profile.json, impact-report.html. Nothing is uploaded with --no-submit.

Troubleshooting

  • 404 on npm — package not published yet; use Path B.
  • EACCES on global install — use a user-owned npm prefix or a Node version manager.
  • command not found: impact — ensure npm’s global bin directory is on your PATH.

Full detail: install-macos.md.
