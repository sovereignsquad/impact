Public shell · MVP scanner

Understand what your machine can run for local AI

IMPACT is a privacy-first CLI that inventories your OS, runtimes, tools, and models — then writes an offline HTML report and JSON profile on your disk. Optional anonymous submission only with your consent.

Install & run Source & docs

Install

Path C (npm) — when published: npm install -g @impact/cli then impact scan. Track: issue #34.

Path B (from source) — always available:

git clone https://github.com/moldovancsaba/impact.git
cd impact
npm ci && npm run build
npm install -g ./apps/cli
impact scan --no-submit -o ./reports

Full detail: install-macos.md (macOS-first; Linux/Windows partial).

What you see after a scan

  • impact-report.html — readable report in your output folder (open in a browser).
  • impact-profile.json — machine-readable impact.v0.3 profile (provenance + confidence on fields).
  • Optional submission preview/receipt files if you opt in and configure an endpoint.

Nothing is sent off your machine by default.

Explore a profile file

Drop impact-profile.json here or choose a file. Parsing happens in your browser — no upload. Validation uses the same @impact/schemas rules as the CLI.

FAQ

Does IMPACT upload my data?

No by default. Scans are local. Submission only happens if you configure an endpoint and explicitly confirm. See the privacy for users page.

Is this a benchmark product?

Not yet. v0.x is discovery and export. Benchmarks are programme roadmap, not current product claims.

Why does npm install say “coming”?

Publishing @impact/cli to the registry is tracked in #34. Until then, use Path B (clone + build) above.
