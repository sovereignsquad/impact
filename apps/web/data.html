Community data

This area shows aggregate views of what the community runs — hardware, tools/runtimes, and models — without exposing individual submissions. When the site is built with a stats API base URL, tables are updated from submitted profiles subject to server privacy thresholds. Otherwise you see honest placeholders only — no fake live numbers.

Status: structure ships first. Live charts and counts appear only when ingest is operational, enough opted-in submissions exist, and privacy thresholds allow publication. See programme issues #48 and #51–#53.
