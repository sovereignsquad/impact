# Impact

**Impact** is an open source system developed and delivered from this repository. The codebase will grow here; this document is the canonical entry point for contributors and users.

- **Repository:** [github.com/moldovancsaba/impact](https://github.com/moldovancsaba/impact)
- **License:** [MIT](LICENSE)

## Project management (roadmap, backlog, todos)

All roadmap items, backlog work, and actionable todos are tracked on the **GitHub Project board**, not in ad-hoc local lists:

**[Impact — roadmap & backlog (Project board)](https://github.com/users/moldovancsaba/projects/2/views/1)**

- Open or discuss scope in **GitHub Issues** in this repository.
- **Prioritize and sequence** work on the project board (Status, milestones, and fields there are authoritative for planning).
- Link issues and pull requests to board items so progress stays visible.

Details: [docs/project-management.md](docs/project-management.md).

## Local checkout

This tree is the delivery location for ongoing work (e.g. `/Users/Shared/projects/impact` on maintainer machines). Clone anywhere you prefer:

```bash
git clone https://github.com/moldovancsaba/impact.git
cd impact
```
